Simple Event plugin that gets you an event post type and methods to handle events.

== Description ==

This plugin was primarely made for developers so they dont always have to start from scratch when implementing events for their clients.
It is desinged to be a framework.
It has the necesarry post type, fields and framework to set you up for success.
Basic usage is through shortcode:

`[atomic-events filter max="10" columns="3" featured="0" skip_first_x="0" skip skip_first_x_featured="1" order="ASC"]`


== Installation ==

1. Upload the Atomic Events plugin to your blog
1. Activate it
1. Create Event post types
1. Place `[atomic-events]` in your templates
